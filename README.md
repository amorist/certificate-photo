
# 证件照 小程序 (原生微信小程序 + 云开发)
欢迎提出改进意见

## 项目仅供学习，如有雷同应用，必究。

用于免费快速生成证件照

- 可自定义照片尺寸、背景颜色 
- 支持换装、换发型 
- 支持手势拖拽缩放编辑图片


- 客服消息自动回复
- 定时触发器
- 云函数海报生成，图片合成

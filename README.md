### 功能

用于免费快速生成证件照

- 可自定义照片尺寸、背景颜色 
- 支持换装、换发型 
- 支持手势拖拽缩放编辑图片


- 客服消息自动回复
- 定时触发器
- 云函数海报生成，图片合成
